## INSTALLATION

### Install Rust

```curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh ```

(See: https://www.rust-lang.org/tools/install)

### Install Tauri 

```cargo install tauri-cli```



(See: https://tauri.app)

### Install Dependencies

In project directory:

```npm install```

### Build Tauri

```cargo tauri build```
